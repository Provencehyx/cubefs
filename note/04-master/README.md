## Master 核心职责

| 职责 | 说明 |
|------|------|
| 集群管理 | 管理整个集群的状态和配置 |
| 节点管理 | DataNode/MetaNode 注册、心跳、故障检测 |
| 拓扑管理 | Zone、NodeSet 管理，故障域隔离 |
| 卷管理 | 创建/删除/扩容卷 |
| 分区管理 | 创建/分裂/迁移 DataPartition/MetaPartition |
| ID 分配 | 全局唯一 ID 分配（分区ID、节点ID等） |
| 负载均衡 | 分区分布、节点负载均衡 |
| 元数据持久化 | 通过 Raft + RocksDB 持久化集群元数据 |
